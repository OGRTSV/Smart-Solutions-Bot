# 🕵️ Smart Solutions Bot — OSINT Telegram-бот

**Автоматизированный Telegram-бот для проверки контрагентов и сбора открытых данных (OSINT) из официальных источников РФ и GitHub.**

Разработан для службы безопасности и HR-отделов как инструмент быстрой проверки физических лиц и организаций.

---

## 🎯 Основные возможности

### 👤 Поиск по ФИО
- Проверка через официальную базу ФНС России (ЕГРЮЛ/ЕГРИП)
- Поиск организаций, где человек является учредителем или руководителем
- Выгрузка результатов в Excel с полной информацией:
  - Название и тип организации (ООО, ИП, ПАО и т.д.)
  - ИНН, КПП, ОГРН
  - Юридический адрес
  - Статус (действующая / ликвидирована / в процессе ликвидации)
  - Роль в организации (учредитель / руководитель)

### 📱 Поиск по номеру телефона
- Поиск организаций по источникам
- Определение компаний, на которые зарегистрирован номер
- Выгрузка в Excel со всеми реквизитами

### 🐙 Анализ GitHub-профиля
- Получение публичной информации о разработчике
- Статистика репозиториев: языки, звёзды, форки, топики
- Анализ активности: коммиты, PR, issues, релизы
- Топ-5 репозиториев по звёздам

### ⚔️ Сравнение двух разработчиков
- Сравнение по 4 метрикам: звёзды, репозитории, подписчики, форки
- Подсчёт очков и определение победителя

### 📜 История запросов
- Сохранение всех запросов в локальную БД
- Пагинация и навигация
- Повторное получение результатов одним кликом
- Экспорт всей истории в Excel

### ⚡ Умное кэширование
- Быстрый ответ из кэша для повторных запросов
- Возможность запустить свежий поиск
- Автоматическое устаревание через 30 дней

## 🛠 Технологии

| Категория | Технологии |
|---|---|
| **Язык** | Python 3.11+ |
| **Telegram-бот** | aiogram 3.7 (асинхронный фреймворк) |
| **HTTP-запросы** | aiohttp, requests |
| **Веб-скрейпинг** | Selenium WebDriver, BeautifulSoup4, lxml |
| **База данных** | SQLite 3 (локальное хранение) |
| **Экспорт данных** | openpyxl (Excel-файлы) |
| **Автоматизация драйверов** | webdriver-manager |

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/OGRTSV/Smart-Solutions-Bot.git
cd Smart-Solutions-Bot
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ
GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
PROXY_URL=  # опционально
```

| Переменная | Описание | Обязательно |
|---|---|---|
| `BOT_TOKEN` | Токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather)) | ✅ Да |
| `GITHUB_TOKEN` | Персональный токен GitHub ([создать здесь](https://github.com/settings/tokens)) — увеличивает лимит с 60 до 5000 запросов/час | ⚠️ Опционально |
| `PROXY_URL` | URL прокси-сервера для обхода блокировок | ⚠️ Опционально |

### 5. Запуск бота
```bash
python main.py
```
